# Action Figure Miles x Swapnice

A landing page website for Action Figure Miles in collaboration with Swapnice.

## Project Structure

```
pages/          - Main website files
backup/         - Backup files
LLM context/    - Context files
```

## Technologies

- HTML/CSS
- Tailwind CSS

## Development

To build the CSS:

```bash
cd backup/backup-1
npm install
npm run build
```

To watch for changes during development:

```bash
npm run watch
```
